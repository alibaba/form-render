# API

## Props

| 属性                  | 描述                                                                      | 类型                 | 默认值     |
| --------------------- | ------------------------------------------------------------------------ | ------------------- | --------- |
| logo                 | logo 模块设置                                                        | [Logo](#Logo)| - | -   |


## Logo

我们将搜索相关的能力放到 `<Search />` 上面配置，包括对应的搜索筛选表单的渲染

| 属性                  | 描述                                                                      | 类型                 | 默认值     |
| --------------------- | ------------------------------------------------------------------------ | ------------------- | --------- |
| image         |   图片地址                                                                        | `string`      |   -     |
| href          |   图片点击跳转地址                                                                  | `string`    |   -     |
| title         |   标题                                                                            | `string`     |   -     |
